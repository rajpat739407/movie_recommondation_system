# 🎬 Movie Recommendation System

A complete movie recommendation system with Streamlit web interface.

## 🚀 Features
- Collaborative Filtering (User-based, Item-based)
- Matrix Factorization (SVD)
- Streamlit Web Interface
- Movie Search & Recommendations

## 📁 Setup
1. Clone repo: `git clone https://github.com/rajpat739407/movie_recommondation_system.git`
2. Install: `pip install -r requirements.txt`
3. Download MovieLens data to `data/` folder
4. Run notebooks in order to train models
5. Launch: `streamlit run app.py`

## 🗂️ Project Structure
- `app.py` - Web interface
- `src/` - ML algorithms  
- `notebooks/` - Training code
- `data/` - Add your dataset here
- `models/` - Generated by training
